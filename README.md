# Readstr

A cross-platform RSVP  speed reading app for iOS, Android, and Desktop. Words flash at your chosen WPM with an anchor-letter focus point to guide the eye. Load a plain text file, EPUB, PDF, or paste text directly.



## Project Status
Android tested and works 

## Development
Built with the [RMP (Rust Multiplatform)](https://github.com/nickthecook/rmp) architecture — shared Rust core, native UI layers.
